# beryl

Beryl is a pixel image to ASCII art converter.


## Usage

```kotlin
val converter = Converter(
        File("path/to/your/image.jpg").inputStream(),
        LightnessApproximator(CharScale.BASE_9),
        FontSize(8, 14))

print(converter.toAscii())
```
